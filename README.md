# dotfiles

setting dotfiles

## Setup script

- install dotfiles

Backup old dotfiles and set symlink dotfiles

```console
$ ./dotfiles.sh install
```

- set symlink dotfiles

```console
$ ./dotfiles.sh set
```

- unlink dotfiles

```console
$ ./dotfiles.sh unset
```

- delete dotfiles

```console
$ ./dotfiles.sh delete
```

- dotfiles list

```console
$ ./dotfiles.sh list
```
